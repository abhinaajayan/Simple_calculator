About This Calculator

This is a Scientific Calculator Web Application built using Flask (Python). It allows users to perform both basic and advanced mathematical operations easily through a clean interface.

Features:

  • Basic operations: Addition, Subtraction, Multiplication, Division
  • Scientific functions: sin, cos, tan (in degrees)
  • Logarithmic and square root calculations
  • Power calculations (x^y)
  • Simple and user-friendly interface

This project is useful for learning web development using Flask and integrating Python's mathematical capabilities into web apps.
